# 🎯 AI Career Advisor

An AI-powered Career Advisor built using **n8n**, **OpenAI GPT-4.1 Mini**, and **Simple Memory**. It provides personalized career guidance, learning roadmaps, skill recommendations, and interview preparation advice.

## 🚀 Features

- 🎯 Personalized Career Guidance
- 📚 AI-Generated Learning Roadmaps
- 💼 Career Path Recommendations
- 🛠️ Skill Gap Analysis
- 🎓 Certification Suggestions
- 🧠 Conversational Memory
- ⚡ Workflow Automation with n8n

## 🛠️ Tech Stack

- n8n
- OpenAI GPT-4.1 Mini
- Simple Memory

## 📦 Installation

1. Download the `ai-career-advisor.json` workflow.
2. Import it into n8n.
3. Configure your OpenAI Chat Model.
4. Save and activate the workflow.

## 🎯 Example Prompts

- I am a 2nd year ECE student. Which AI career is best for me?
- Create a roadmap to become an AI Engineer.
- What skills should I learn for AI jobs?
- Suggest projects to improve my portfolio.
